Essential before drafting the Option α 1-page proposal. These define the DR-learner / DML / OPE machinery the project extends.

  • Kennedy 2023 review (arXiv:2203.06469) — "Semiparametric Doubly Robust Targeted Double Machine Learning: a Review". ~60-page review of DR estimation, EIF, cross-fitting, semiparametric efficiency. Single best on-ramp for Option α. VERIFY arXiv ID before formally citing.
